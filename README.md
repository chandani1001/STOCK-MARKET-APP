# STOCK-MARKET-APP
Stock Market Analysis &amp; Prediction Dashboard built with Python, Machine Learning, and Streamlit. Performs trend analysis, moving averages, volatility, and correlation on AAPL, MSFT, NFLX, and GOOG, with a basic model for forecasting stock price behavior.
[Click here to view the live dashboard](https://stock-market-appp.streamlit.app/)
